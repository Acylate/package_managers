poetry new - создание базовой структуры проекта
poetry init - инициализирует poetry в текущем проекте

=====================

poetry add [библиотека] - установить пакет/ы (библиотеки)
poetry add --dev [библиотека] - установка пакета для разработки

poetry remove [библиотека] - удаление библиотеки
poetry install [библиотека] - установка пакетов из pyproject.toml
poetry show (--tree) - посмотреть все пакеты проекта (вместе с --tree то удобней)

poetry run <команда для запуска> - запуск из виртуального окружения
poetry run <кастомные скрипты/функции> - запуск скрипта из pyproject.toml (прописывать самому в [tool.poetry.scripts])

======================

poetry update - обновление пакетов
poetry build - делает программу в .exe файл

poetry config --list - просмотр текущего конфига приложения
poetry config (переменная) (значение) - изменить значение переменной в конфиге
