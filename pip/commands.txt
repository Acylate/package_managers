python -m venv .venv - создание виртуального окружения
.venv\Scripts\activate - активация виртуального окружения (windows)
source .venv\bin\activate - активация виртуального окружения (linux)

===================================
touch requirements.txt - создать requirements.txt (Linux) (список библиотек с версиями)
===================================

pip install [библиотека] [библиотека2] - устанавливает библиотеку/и на новейшей версии
pip install -r requirements.txt - для развертывания проекта (установка библиотек из requirements.txt)
pip uninstall [библиотека] [библиотека2] - удаляет библиотеку/и

pip freeze - показывает список скачанных библиотек с их версиями как requirements.txt
pip list - вывод установленных библиотек просто

===================================
Заполнение requirements.txt

Не заполнять с помощи команды "pip freeze > requirements.txt"!!! Добавятся больше библиотек для других библиотек
Заполнять вручную, брать библиотеки из pip freeze
pip install [библиотека] > pip freeze (копируем потом из консоли библиотеку с версией и вставляем в requirements.txt)
